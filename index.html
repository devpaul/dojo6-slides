The Dojo Framework

"Batteries-included" reactive framework

  • Dojo toolkit was introduced in 2005
  • Dojo Framework was a complete rewrite: TypeScript, reactive

Features

  • Widgets
  • Routing
  • Processes
  • State framework
  • Polyfills
  • Custom Elements
  • Build time rendering
  • Internationalization
  • Accessibility

Dojo's Reactive Lifecycle

  • Widgets: written in JSX or hyperscript, render the view
  • Process: operates the business logic of the application; mutates the store
  • State: Globally held by the store
