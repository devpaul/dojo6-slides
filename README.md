# Dojo 6: A TypeScript-first reactive framework

Created by Paul Shannon [@developerPaul](https://twitter.com/developerPaul).

## Quick Start

* `npm install`
* `npm run build`
* `npm start`
* open [Presentation @ http://localhost:8765](http://localhost:8765)

## Usage

* Press `s` to open speaker notes
